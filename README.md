# AK Gesture

AI Powered Hand Gesture Recognition Platform built using Python, Flask, OpenCV and MediaPipe.

## Features

- Hand Detection
- Finger Counter
- Air Canvas
- Virtual Mouse
- Rock Paper Scissors
- Feature Switching

## Tech Stack

- Python
- Flask
- OpenCV
- MediaPipe
- HTML
- CSS
- JavaScript

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/AK-Gesture.git

cd AK-Gesture

pip install -r requirements.txt

python app.py
